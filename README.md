# My Cypress Automation

This project Cypress to automate the testing of a login feature and user update feature. Cypress is a JavaSchipt-base end-to-end testing framework that framework tha allows developers to write tests and execute the in a real browser.

## Installation
```bash
npm install
```
> **NOTE:**
> >
> it is necessary to have **node** installed

## Running
```bash
# You open the Cypress UI
```
npx cypress open

# Run the automation by commnad line
npx cypress run
